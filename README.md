# My personal MERN setup for fast start developing fullstack apps

***
## Under the hood:
1. Webpack
2. Babel (ES6/ES7 support)
3. MongoDB
4. Node
5. React
6. Hot Reloading
7. Express

***
## How to make it work?
1. Clone this repo
2. Run `npm install` in terminal
3. Run `npm run development`
4. Go to `http://localhost:3000`
